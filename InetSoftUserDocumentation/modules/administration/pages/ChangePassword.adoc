Summary: How to change the administrator password in Enterprise Manager. See CreateUser for changing a user password.
endif::ai[]

To change your administrator password, press the {buttonUserOptions} at the top of Enterprise Manager and select ‘Change Password’.

image::ChangePassword.png[alt="change password dialog with old and new fields"]

Enter the old password and new password, and press ‘Change Password’. To change a user’s password, see xref:CreateUser.adoc[].
