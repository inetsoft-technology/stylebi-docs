The following data type constants are available.

== Type
xref:commonscript:StyleConstant.adoc[] object:: One of the (integer) constants below:
+
[cols="1,5"]
|===
|Type|Description

|BOOLEAN|`true` or `false`
|BYTE|one byte number (8-bit integer) in range  -128 to 127
|CHAR|character
|DATE|date with no time component
|DOUBLE|64-bit number
|ENUM|user-defined string constants
|FLOAT|32-bit number
|INTEGER|32-bit signed integer  in range -2,147,483,648 to 2,147,483,647
|LONG|64-bit signed integer
|SHORT|16-bit integer in range -32,768 to 32,767
|STRING|text string
|TIME_INSTANT|time and date
|===

== Example
[source,javascript]
RangeSlider1.rangeType = StyleConstant.DATE;
