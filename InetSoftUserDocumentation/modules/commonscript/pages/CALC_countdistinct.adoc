Summary: Documents the CALC.countDistinct(array) function, which counts the number of distinct elements in the given array (numeric or non-numeric). Returns a number. Applies to both Dashboard script and Data Worksheet script contexts.

Counts the number of distinct elements in array.

== Parameters
array::			Array of numeric or non-numeric values



== Example
[source,javascript]
----
CALC.countDistinct([3,3,2,4,4,5,6]);  // returns 5
----

// Column of Dashboard table TableView1:

----
