ifdef::ai[]
Summary: Documents the CALC.averagea(array) function, which returns the average (arithmetic mean) of an array of numerical, text, and/or Boolean values. Text evaluates to 0, true evaluates to 1, and false evaluates to 0. Applies to both Dashboard script and Data Worksheet script contexts.
endif::ai[]

Returns the average (arithmetic mean) of the arguments.  {CALCExcelFunctionPage} Text and Boolean values are evaluated as follows:
[square]
* Text evaluates to `0`
* `true` evaluates to `1`
* `false` evaluates to `0`

== Parameters
array::			Array of numerical, text, and/or Boolean values to average


== Example
[source,javascript]
----
CALC.averagea([1,2,3,4,'asd',true,false]);  // returns 1.5714285714285714
----
