Summary: Documents the CALC.average(array) function, which returns the arithmetic mean of an array of numerical values. Takes a single parameter: an array of real values. Applies to both Dashboard script and Data Worksheet script contexts. See also CALC.averagea() for a variant that also processes text and Boolean values.
endif::ai[]

[WARNING,caption=see also]
xref:CALC_averagea.adoc[] to process text and Boolean values in addition to numerical values.

Returns the average (arithmetic mean) of the data in array. {CALCExcelFunctionPage}

== Parameters
array::			Array of real values to average


== Example
[source,javascript]
----
CALC.average([4,5,6,7,5,4,3]);  // returns 4.857142857142857
----
