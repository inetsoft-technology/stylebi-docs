Summary: How to create a Circle Packing chart in Visual Composer, which represents a measure using hierarchically nested circles. Place hierarchical dimensions in the T region and the measure in the Size region; use the Color region to distinguish groups. CirclePackingChart, TreemapChart, SunburstChart, and IcicleChart all bind identically (T/Size/Color) and encode the same hierarchical part-to-whole data — they differ only in layout: Circle Packing uses nested circles, Treemap uses nested rectangles, Sunburst uses radial wedges, and Icicle uses layered rectangular bands (the Cartesian equivalent of Sunburst). MarimekkoChart shows a related but structurally different 2-level mosaic (X/Y/Color/Break By binding, not T/Size).
endif::ai[]

A  circle packing chart displays a *measure* by using circles in a hierarchical display.

include::ROOT:partial$measureAndDimensionTip.adoc[]

image::commonscript:circlePackingConfig1.png[alt="circle packing chart example"]

To create a circle packing chart, follow the basic steps below:

include::partial$chartNewbieTip.adoc[]

. Press the {buttonSelectChartStyle}. Choose the ‘Circle Packing’ style. Press the {buttonApply}.
+
image::commonscript:circlePackingConfig2.png[alt="select Circle Packing chart style"]

. From the ‘Dimensions’ folder of the *Data Source* panel, drag the desired hierarchical *dimension* fields to the ‘T’ region. Add dimensions from the outermost to the innermost level of the hierarchy.
+
include::ROOT:partial$dimensionTip.adoc[]
+
image::commonscript:circlePackingConfigT.png[alt="drag hierarchical dimensions to T region"]

. From the ‘Measures’ folder of the *Data Source* panel, drag a desired *measure* to the ‘Size’ region.
+
include::ROOT:partial$measureTip.adoc[]
+
image::commonscript:circlePackingConfigSize.png[alt="drag measure to Size region"]

. *_Optional:_* Drag a dimension from the *Data Source* panel to the ‘Color’ region to distinguish dimension values by color.
+
image::commonscript:circlePackingConfig3.png[alt="drag dimension to Color region"]

. *_Optional:_* Press the ‘Edit Color’ button next to the dimension, and choose the desired colors to represent the dimension values. Press the {buttonApply}.
+
image::commonscript:EditColorButton.png[alt="press Edit Color button"]

. *_Optional:_* To fill any of the larger circles with static color (e.g., the outermost containing circles), click a circle to select it, and use the ‘Background’ property on the *Format* tab to apply a color. See xref:VisualFormat.adoc[] for more information about formatting.

. *_Optional:_* To apply a color to the circle lines, right-click the Chart and select ‘Properties’. Set the ‘Border Color’ under the *Advanced* tab of the ‘Properties’ dialog box. See xref:ChartProperties.adoc[] for more information about properties.

. *_Optional:_* {multipleDimensionNoColor}

. {finishEditing}
+
image::commonscript:circlePackingConfig1.png[alt="completed circle packing chart"]

NOTE: To create a grid of charts, place dimensions into the ‘X’ and/or ‘Y’ regions as well. See xref:TrellisChartGrid.adoc[] for more information.
