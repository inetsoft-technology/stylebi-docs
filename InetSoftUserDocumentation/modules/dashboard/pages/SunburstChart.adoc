Summary: Reference for creating a Sunburst chart in Visual Composer, which displays a measure using hierarchically arranged wedges in a radial layout (like a multi-level pie chart). Place hierarchical dimensions in the T region and the measure in the Size region. Compare to TreemapChart (rectangles) and CirclePackingChart (circles).
endif::ai[]

A  Sunburst (mosaic) chart displays a *measure* by using wedges in a hierarchical display.

include::ROOT:partial$measureAndDimensionTip.adoc[]

image::commonscript:SunburstConfig1.png[alt="sunburst chart example"]

To create a sunburst chart, follow the basic steps below:

include::partial$chartNewbieTip.adoc[]

. Press the {buttonSelectChartStyle}. Choose the ‘Sunburst’ style. Press the {buttonApply}.
+
image::commonscript:SunburstConfig2.png[alt="press Select Chart Style, choose Sunburst"]

. From the ‘Dimensions’ folder of the *Data Source* panel, drag the desired hierarchical *dimension* fields to the ‘T’ region. Add dimensions from the outermost to the innermost level of the hierarchy.
+
include::ROOT:partial$dimensionTip.adoc[]
+
image::commonscript:SunburstConfigT.png[alt="drag hierarchical dimensions to T region"]

. From the ‘Measures’ folder of the *Data Source* panel, drag a desired *measure* to the ‘Size’ region.
+
include::ROOT:partial$measureTip.adoc[]
+
image::commonscript:SunburstConfigSize.png[alt="drag measure to Size region"]

. *_Optional:_* Drag a dimension from the *Data Source* panel to the ‘Color’ region to distinguish dimension values by color.
+
image::commonscript:SunburstConfig3.png[alt="drag dimension to Color region"]

. *_Optional:_* Press the {buttonEditMeasure} next to the measure, and select the desired aggregation method.

. *_Optional:_* {multipleDimensionNoColor}

. {finishEditing}
+
image::commonscript:SunburstConfig1.png[alt="completed sunburst chart"]

NOTE: To create a grid of charts, place dimensions into the ‘X’ and/or ‘Y’ regions as well. See xref:TrellisChartGrid.adoc[] for more information.
