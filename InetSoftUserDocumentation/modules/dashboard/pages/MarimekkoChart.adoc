A Marimekko chart (also known as a Mekko or mosaic chart) displays a *measure* by using the sizes of rectangles in a hierarchical display.

To create a Marimekko chart, place a *dimension* in the ‚ÄòX‚Äô region of the Chart Editor to create the X-axis labels, and place the desired Mekko category *dimension* in the ‚ÄòColor‚Äô or ‚ÄòBreak By‚Äô region of the Chart Editor. The Mekko category dimension determines the grouping in each column of the chart.

NOTE: To create a grid of charts, place dimensions into the ‚ÄòX‚Äô and/or ‚ÄòY‚Äô regions as well. See xref:TrellisChartGrid.adoc[] for more information.

. Create a new Dashboard based on the ‚ÄòSales Explore‚Äô Data Worksheet.
