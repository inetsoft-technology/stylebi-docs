// metadata for AI
ifdef::ai[]
Reference for creating an Icicle chart in Visual Composer, which displays a measure using hierarchically sized rectangles in a hierarchical layout. Place hierarchical dimensions in the T region and the desired measure in the Size region. Use the Color region to distinguish dimension values by color. To create a trellis grid of icicle charts, place dimensions in the X and/or Y regions. Similar to a Treemap chart but with a different layout orientation.
endif::ai[]

An icicle chart displays a measure by using the sizes of rectangles in a hierarchical display.

include::ROOT:partial$measureAndDimensionTip.adoc[]

image::commonscript:icicleConfig1.png[alt="icicle chart example"]

To create an icicle chart, follow the basic steps below:

include::partial$chartNewbieTip.adoc[]

. Press the {buttonSelectChartStyle}. Choose the ‘Icicle’ style. Press the {buttonApply}.
+
image::commonscript:icicleConfig2.png[alt="press Select Chart Style button"]

. From the ‘Dimensions’ folder of the *Data Source* panel, drag the desired hierarchical *dimension* fields to the ‘T’ region. Add dimensions from the outermost to the innermost level of the hierarchy.
+
include::ROOT:partial$dimensionTip.adoc[]
+
image::commonscript:icicleConfigT.png[alt="drag hierarchical dimensions to T region"]

. From the ‘Measures’ folder of the *Data Source* panel, drag a desired *measure* to the ‘Size’ region.
+
include::ROOT:partial$measureTip.adoc[]
+
image::commonscript:icicleConfigSize.png[alt="drag measure to Size region"]

. *_Optional:_* Drag a dimension from the *Data Source* panel to the ‘Color’ region to distinguish dimension values by color.
+
image::commonscript:icicleConfig3.png[alt="drag dimension to Color region"]

. *_Optional:_* Press the ‘Edit Color’ button next to the dimension, and choose the desired colors to represent the dimension values. Press the {buttonApply}.
+
image::commonscript:EditColorButton.png[alt="press Edit Color button"]

. *_Optional:_* {multipleDimensionNoColor}

. {finishEditing}
+
image::commonscript:icicleConfig1.png[alt="completed icicle chart"]

NOTE: To create a grid of charts, place dimensions into the ‘X’ and/or ‘Y’ regions as well. See xref:TrellisChartGrid.adoc[] for more information.
