= Egraph.setY2TitleSpec(spec)
:navtitle: Egraph.setY2TitleSpec()
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]






Specifies the Y-axis title information for Y-axis to the *right* of the  Chart.  The corresponding “getter” function is xref:chartAPI:EGraph_getY2TitleSpec.adoc[].

== Parameter
spec::			A xref:chartAPI:TitleSpec.adoc[] object







== Example

.Chart Component Script
[source,javascript]
--
dataset = [["State", "Quantity"], ["NJ", 200], ["NY", 300]]; //<1>
graph = new EGraph();
var elem = new IntervalElement("State", "Quantity");
var spec = new TitleSpec(); //<2>
spec.setLabel("Y Title");
graph.setY2TitleSpec(spec); //<3>
graph.addElement(elem);
--
<1> {dataset}
<2> Create the xref:TitleSpec.adoc[] object.
<3> Use `setY2TitleSpec` to add the `TitleSpec` to Chart.

[TIP]
=====

To change the property on a Chart that was previously created with the Chart Editor, use xref:chartAPI:EGraph_getY2TitleSpec.adoc[] ① to get a handle to the `TitleSpec` object. For example:


[source,javascript]
--
var spec = graph.getY2TitleSpec();
spec.setLabel("Y Title");
--
