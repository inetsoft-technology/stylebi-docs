Specifies whether *tick marks* are displayed at positions where duplicate labels have been suppressed on a xref:TimeScale.adoc[]. When a date format produces the same text for multiple consecutive ticks (for example, formatting monthly data as `'yyyy'` causes every month in the same year to display the same label), the duplicate labels are hidden. This property controls whether tick marks are still drawn at those hidden-label positions.  {noGetter}

== Parameter
Boolean:: `true`: show tick marks at all positions, including those with suppressed duplicate labels +
`false`: hide tick marks where duplicate labels have been suppressed

<1> {dataset} 90 daily data points (Jan–Mar 2023). `setType(DAY)` and `setIncrement(1)` force one tick per day; `setMaxTicks(100)` allows all 90 ticks to be generated.
<2> Create the `AxisSpec` object. Formatting as `'MMM'` means every day in the same month shares the same label — 89 out of 90 are duplicates.
<3> Use `setAllTicks(true)` to show all 90 daily tick marks even where duplicate labels are hidden. Change to `false` to see only 3 tick marks (one per month).
