= LegendSpec.setPreferredSize(value)
:navtitle: LegendSpec.setPreferredSize()
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]

[WARNING,caption=See Also]
====
[square]
* xref:chartAPI:EGraph_setLegendLayout.adoc[] to place the legend in a predefined location.
====


Specifies the legend size (pixels) for cases when xref:EGraph_setLegendLayout.adoc[] is set to `Chart.IN_PLACE`.  The corresponding “getter” function is `getPreferredSize()`.

== Parameter
value::			a {javaawtDimension} object





== Example


.Chart Component Script
[source,javascript]
--

dataset = [["State", "Quantity"], ["NJ",200], ["NY",300]]; //<1>
graph = new EGraph();
var elem = new IntervalElement("State", "Quantity");
var frame = new CategoricalColorFrame();
frame.setField("State");
var spec = new LegendSpec(); //<2>
spec.setPosition(java.awt.Point(50,245)); // in pixels
spec.setPreferredSize(java.awt.Dimension(100,40)); //<3>
frame.setLegendSpec(spec); //<4>
elem.setColorFrame(frame);
graph.setLegendLayout(Chart.IN_PLACE);
graph.addElement(elem);

--
<1> {dataset}
<2> Create the xref:LegendSpec.adoc[] object.
<3> Use `setPreferredSize` to assign the size.
<4> Use xref:VisualFrameSetLegendSpec.adoc[] to assign the `LegendSpec` object to the `ColorFrame`.

[TIP]
=====





To change the property on a Chart that was *previously created* with the Chart Editor, use “getter” methods such as xref:chartAPI:EGraph_getElement.adoc[] ① and xref:chartAPI:GraphElementGetColorFrame.adoc[] ② to obtain a handle to the desired xref:chartAPI:GraphElement.adoc[] and xref:chartAPI:ColorFrame.adoc[]. For example:

[source,javascript]
--
var elem = graph.getElement(0); <1>
var frame = elem.getColorFrame(); <2>
// Compact syntax: var frame = graph.getElement(0).getColorFrame();

var spec = new LegendSpec();
spec.setPosition(java.awt.Point(50,245)); // in pixels
spec.setPreferredSize(java.awt.Dimension(100,40));
frame.setLegendSpec(spec);
graph.setLegendLayout(Chart.IN_PLACE);

--
