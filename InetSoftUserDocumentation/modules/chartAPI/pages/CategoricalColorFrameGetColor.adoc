Returns the color assigned to the specified value, `val`.  You can also use the `bindingInfo.colorFrame.getColor()` method syntax in *onRefresh* script.  The corresponding РђюsetterРђЮ function is xref:CategoricalColorFrameSetColor.adoc[].

== Parameter
val::			a data value







[#Example1]
== Example 1

This example illustrates how to use xref:dashboardscript:ChartBindingInfo.adoc[] properties to control the xref:ColorFrame.adoc[].

. Bind a xref:dashboard:BarChart.adoc[]  to the sample РђўAll SalesРђЎ Data Worksheet, with РђўCompanyРђЎ (top 5) on the X-axis, and Sum(Total) on the Y-axis.
+
include::ROOT:partial$AllSalesQuery.adoc[]


. Add the following script in the *onRefresh* handler. (See xref:dashboardscript:AddScriptToDashboard.adoc#AddDashboardScript[Add Dashboard Script].)

+
.onRefresh Script
[source,javascript]
----
Chart1.bindingInfo.setColorField("Employee",Chart.STRING);
Chart1.bindingInfo.colorFrame = new CategoricalColorFrame;
Chart1.bindingInfo.colorFrame.setColor("Robert",java.awt.Color.lightGray);
var robColor = Chart1.bindingInfo.colorFrame.getColor("Robert");
Chart1.bindingInfo.colorFrame.setColor("Sue",robColor);

----

[#Example2]
== Example 2

.Chart Component Script
[source,javascript]
----
dataset = [["State","Quantity"],["NJ",200],["NY",300]]; //<1>
graph = new EGraph();
var elem = new IntervalElement("State", "Quantity");
frame = new CategoricalColorFrame(); //<2>
frame.setField("State");
frame.setColor('NJ',java.awt.Color(0xff0000));
var NJcolor = frame.getColor('NJ'); //<3>
frame.setColor('NY',NJcolor); //<4>
elem.setColorFrame(frame); //<5>
graph.addElement(elem);

----
<1> {dataset}
<2> Create the xref:ColorFrame.adoc[] object.
<3> Use `getColor` to obtain the color for '`NJ`'.
<4> Use `setColor` to assign the '`NJ`'  color to '`NY`'.
<5> Use xref:GraphElementSetColorFrame.adoc[] to add the `ColorFrame` to the Chart.
