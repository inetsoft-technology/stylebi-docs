= SizeFrame.setSmallest(value)
:navtitle: SizeFrame.setSmallest()
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]

[CAUTION,caption=inherits from]
====
[square]
* xref:VisualFrame.adoc[]
====

Specifies the lowest value in the range. For xref:PointElement.adoc[] and xref:LineElement.adoc[] graphs, this is the size (in pixels) at which the smallest value in the bound column is displayed. Larger values are scaled according to the mapping defined by the object’s scaling method (e.g., linear for xref:LinearSizeFrame.adoc[], etc.). For xref:SchemaElement.adoc[] and xref:IntervalElement.adoc[] graphs, the value specified for the `smallest` property is relative to the xref:SizeFrameSetMax.adoc[] value.

For example, if `smallest=50` and `max=100`, the smallest value in the bound column is displayed at one half of the maximum allowable size. You can also set this using the `bindingInfo.sizeFrame.smallest`  property syntax in *onRefresh* script. See xref:#Example2[] below.  The corresponding “getter” function is `getSmallest()`.



== Parameter
value::			Number of pixels, or relative size



== Example 1

.Chart Component Script
[source,javascript]
--
dataset = [["State", "Quantity", "width"], ["NJ", 200, 5],["NY", 300,10], ["PA",75,15]]; //<1>
graph = new EGraph();
var elem = new IntervalElement("State", "Quantity");
var frame = new LinearSizeFrame(); //<2>
frame.setField("width"); //<3>
frame.setSmallest(10); //<4>
frame.setLargest(50);
frame.setMax(100);
elem.setSizeFrame(frame); //<5>
graph.addElement(elem);
--
<1> {dataset}
<2> Create the `LinearSizeFrame` object.
<3> Use the inherited  xref:VisualFrameSetField.adoc[] property to assign the desired measure to the `LinearSizeFrame`.
<4> Use `setSmallest` to display the largest value at 50 pixels.
<5> Use xref:GraphElementSetSizeFrame.adoc[] to add the `LinearSizeFrame` to the xref:IntervalElement.adoc[].

[TIP]
=====


To change the property on a Chart that was *previously created* with the Chart Editor, use “getter” methods such as xref:EGraph_getElement.adoc[] {circle1} and xref:GraphElementGetSizeFrame.adoc[] {circle2}  to obtain a handle to the desired xref:GraphElement.adoc[] and xref:SizeFrame.adoc[]. For example:

[source,javascript]
--
var elem = graph.getElement(0); //<1>
var frame = elem.getSizeFrame(); //<2>
frame.setSmallest(10);
frame.setLargest(50);
frame.setMax(100);
--
=====
