= AxisSpec.setLastOrAll(Boolean)
:navtitle: setLastOrAll()
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]

Specifies the truncation behavior for multi-line axis labels when there is insufficient space to display all lines. When `true`, only the *last line* of the label is displayed; when `false`, as many lines as possible are displayed . {noGetter}

== Parameter
Boolean:: `true`: keep only the last line of a truncated multi-line label +
`false`: keep as many lines as can fit
