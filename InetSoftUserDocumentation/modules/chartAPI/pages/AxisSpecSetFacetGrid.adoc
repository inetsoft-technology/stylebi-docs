= AxisSpec.setFacetGrid(Boolean)
:navtitle: setFacetGrid()
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]




Specifies whether axis grid lines extend across axis labels (as in a facet table layout). When `true`, grid lines extend beyond the plot border across the label region. When `false`, grid lines are clipped at the plot border.

NOTE: This is distinct from xref:chartAPI:FacetCoordSetFacetGrid.adoc[], which controls whether a border is drawn around the outer perimeter of a facet chart.

== Parameter
Boolean:: `true`: grid lines extend across labels (beyond plot border) +
`false`: grid lines stop at plot border (default)
