[CAUTION,caption=inherits from]
====
[square]
* xref:VisualFrame.adoc[]
====

Specifies the static texture for graphical elements in element script . If the data in the column assigned by the inherited xref:VisualFrameSetField.adoc[] function are `GTexture` numbers, these data values are used instead of the `setTexture()` setting. You can also set this using the `bindingInfo.textureFrame.texture`  property syntax in *onRefresh* script . See xref:#Example2[] below.  The corresponding ÔÇťgetterÔÇŁ function is `getTexture()`.

== Example 1

.Chart Component Script
[source,javascript]
----
dataset = [["State","Quantity"], ["NJ",300], ["NY",200]];  //<1>
graph = new EGraph();
var elem = new IntervalElement("State", "Quantity");
frame = new StaticTextureFrame(); //<2>
frame.setTexture(GTexture.PATTERN_18);  //<3>
elem.setTextureFrame(frame); //<4>
graph.addElement(elem);
----
<1> {dataset}
<2> Create the `StaticTextureFrame` object.
<3> Use `setTexture` to assign `GTexture.PATTERN_18`.
<4> Use xref:GraphElementSetTextureFrame.adoc[] to add the `TextureFrame` to the xref:IntervalElement.adoc[].

[TIP]
=====

To change the property on a Chart that was *previously created* with the Chart Editor, use ÔÇťgetterÔÇŁ methods such as xref:EGraphGetElement.adoc[] {circle1} and xref:GraphElementGetTextureFrame.adoc[] {circle2} to obtain a handle to the desired xref:GraphElement.adoc[] and xref:TextureFrame.adoc[]  objects. For example:

[source,javascript]
----
var elem = graph.getElement(0); //<1>
var frame = elem.getTextureFrame(); //<2>
frame.setTexture(GTexture.PATTERN_18);
----
