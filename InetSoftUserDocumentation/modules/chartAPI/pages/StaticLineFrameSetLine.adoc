====
[square]
* xref:VisualFrame.adoc[]
====

Specifies the static line style for graphical elements in element script . If the data in the column assigned to the inherited xref:VisualFrameSetField.adoc[] property are `GLine` objects, these data values are used instead of the `setLine()` value. You can also set this using the `bindingInfo.lineFrame.line`  property syntax in *onRefresh* script . See xref:#Example2[] below.  The corresponding ‚Äúgetter‚Äù function is `getLine()`.

== Parameter
value::			A xref:GLine.adoc[] constant or `Chart.NONE` for empty border:

== Example 1

.Chart Component Script
[source,javascript]
----

dataset = [["State", "Quantity"], ["NJ", 300], ["NY", 200]]; //<1>
graph = new EGraph();
var elem = new PointElement("State", "Quantity");
var frame = new StaticLineFrame(); //<2>
frame.setLine(GLine.DOT_LINE); //<3>
elem.setLineFrame(frame); //<4>
graph.addElement(elem);
----
<1> {dataset}
<2> Create the `LineFrame` object.
<3> Use `setLine` to assign the dotted line style to the `StaticLineFrame`.
<4> Use xref:GraphElementSetLineFrame.adoc[] to add the `LineFrame` to the xref:PointElement.adoc[].

[TIP]
=====
To change the property on a Chart that was *previously created* with the Chart Editor, use ‚Äúgetter‚Äù methods such as xref:EGraphGetElement.adoc[] {circle1} to obtain a handle to the desired xref:GraphElement.adoc[] object. For example:

[source,javascript]
----
var elem = graph.getElement(0); //<1>
var frame = new StaticLineFrame();
frame.setLine(GLine.DOT_LINE);
elem.setLineFrame(frame);
----
