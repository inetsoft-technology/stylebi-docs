Returns the chart legend’s height or width in pixels set by xref:chartAPI:EGraphSetLegendPreferredSize.adoc[]. The corresponding “setter” function is xref:chartAPI:EGraphSetLegendPreferredSize.adoc[].



== Returns
Integer (pixels)

== Example
.Chart Component Script
[source,javascript]
----
dataset = [["State", "Quantity"], ["NJ",200], ["NY",300]]; //<1>
graph = new EGraph();
var elem = new IntervalElement("State", "Quantity");
var frame = new CategoricalColorFrame();
frame.setField("State");
elem.setColorFrame(frame);
graph.setLegendPreferredSize(150); //<2>
graph.addElement(elem);
alert(graph.getLegendPreferredSize()); //<3>
----
<1> {dataset}
<2> Set the legend size.
<3> Use `getLegendPreferredSize()` to get the legend size.
