Adds the specified xref:chartAPI:GraphElement.adoc[] to the chart. See xref:chartAPI:ChartElements.adoc[] for available elements.  The inverse operations are xref:EGraphRemoveElement.adoc[] and xref:EGraphClearElements.adoc[].


== Parameter
elem::			a xref:chartAPI:GraphElement.adoc[] object



== Example


.Chart Component Script
[source,javascript]
----
dataset = [["State", "Quantity"], ["NJ", 200], ["NY", 300]]; //<1>
graph = new EGraph();
var elem = new LineElement("State", "Quantity");
graph.addElement(elem); //<2>
----
<1> {dataset}
<2> Use `addElement` to add the LineElement to Chart.
