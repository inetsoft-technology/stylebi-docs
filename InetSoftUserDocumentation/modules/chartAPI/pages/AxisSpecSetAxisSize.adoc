= AxisSpec.setAxisSize(value)
:navtitle: setAxisSize()
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]

Specifies a fixed size for the axis area in pixels. For a vertical axis, this sets the width; for a horizontal axis, this sets the height. The corresponding getter function is `getAxisSize()`.

== Parameter
value:: Fixed size of the axis area in pixels
