= EGraph.getX2TitleSpec()
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]





Returns a xref:chartAPI:TitleSpec.adoc[] object containing title information for the X-axis *above* the chart.  The corresponding “setter” function is xref:chartAPI:EGraph_setX2TitleSpec.adoc[].






== Returns
A xref:chartAPI:TitleSpec.adoc[] object




== Example

.Chart Component Script
[source,javascript]
--
var spec = Chart1.graph.getX2TitleSpec();
alert(spec.getLabel());
--
