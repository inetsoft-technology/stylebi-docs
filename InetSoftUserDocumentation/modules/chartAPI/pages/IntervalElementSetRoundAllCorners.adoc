= IntervalElement.setRoundAllCorners(Boolean)
:navtitle: setRoundAllCorners()
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]

Specifies whether all four corners of each bar are rounded equally, or only the open (value) end. Requires a non-zero corner radius set via xref:IntervalElementSetCornerRadius.adoc[]. Not available for stacked bar charts. {noGetter}

== Parameter
Boolean:: `true`: round all four corners +
`false`: round only the open (value) end (default)

[TIP]
=====
To change the property on a Chart that was *previously created* with the Chart Editor, use xref:EGraphGetElement.adoc[] to obtain a handle to the desired xref:GraphElement.adoc[] object. For example:

[source,javascript]

elem.setCornerRadius(0.5);
elem.setRoundAllCorners(true);
