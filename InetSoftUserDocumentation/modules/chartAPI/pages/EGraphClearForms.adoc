= EGraph.clearForms()
:navtitle: clearForms()
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]

[WARNING,caption=See Also]
====
[aquare]
* xref:chartAPI:RepresentMultipleMeasures.adoc[] to add multiple elements.
====

Removes all `GraphForm` objects from the chart. The inverse operation is xref:EGraphAddForm.adoc[].





== Example


.Chart Component Script
[source,javascript]
--
dataset = [["State", "Quantity"], ["NJ", 200], ["NY", 300]]; //<1>
graph = new EGraph();
var elem = new IntervalElement("State", "Quantity");
var form = new LineForm();
form.addPoint(new java.awt.Point(0,0));
form.addPoint(new java.awt.Point(100,100));
form.addPoint(new java.awt.Point(200,100));
form.setFill(true);
graph.addForm(form); //<2>
graph.addElement(elem);
graph.clearForms(); //<3>

--
<1> {dataset}
<2> Add a xref:LineForm.adoc[] object.
<3> Use `clearForms` to clear `LineForm`.
