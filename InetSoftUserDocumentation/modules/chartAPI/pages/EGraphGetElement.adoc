Returns a handle to the xref:chartAPI:GraphElement.adoc[] object specified by the index. This is useful in cases where the element was not explicitly assigned to a variable at creation time. To add an element, use xref:EGraphAddElement.adoc[].


== Parameter
index::			Integer index of element

Follow the steps below:

. Create a Dashboard based on the ‘Sales Explore’ Data Worksheet.

----
var elem = graph.getElement(0); <1>
elem.setHint(GraphElement.HINT_SHINE,'false'); <2>
----
<1> Obtain handle to `GraphElement` object.
<2> Apply operations to `GraphElement` object.
