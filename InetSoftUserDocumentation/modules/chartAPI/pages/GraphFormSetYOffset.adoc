= GraphForm.setYOffset(value)
:navtitle: GraphForm.setYOffset()
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]



Specifies an offset (in pixels) by which to shift the xref:GraphForm.adoc[] object vertically. Positive values offset the object upward; negative values offset the object downward.  The corresponding “getter” function is `getYOffset()`.


== Parameter
value::			an integer (pixels)

== Example


.Chart Component Script
[source,javascript]
--
dataset = [["State", "Quantity"],["NJ", 200],["NY", 300]]; //<1>
graph = new EGraph();
var elem = new IntervalElement("State", "Quantity");
var form = new LabelForm(); //<2>
form.setLabel("label1");
form.setValues(['NY', 100]);
form.setXOffset(-50); //<3>
form.setYOffset(100); //<4>
graph.addForm(form); //<5>
graph.addElement(elem);
--
<1> {dataset}
<2> Create the  xref:GraphForm.adoc[] object.
<3> Use `setXOffset()` to set the horizontal offset.
<4> Use `setYOffset()` to set the vertical offset.
<5> Use xref:EGraphAddForm.adoc[] to add the `GraphForm` to the Chart.
