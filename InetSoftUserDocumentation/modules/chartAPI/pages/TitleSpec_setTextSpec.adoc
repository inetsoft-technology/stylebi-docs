= TitleSpec.setTextSpec(spec)
:navtitle: TitleSpec.setTextSpec()
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]




Specifies the title text attributes, such as color, font, format, etc.  The corresponding “getter” function is xref:TitleSpec_getTextSpec.adoc[].

== Parameter
spec:: a xref:chartAPI:TextSpec.adoc[] object



== Example

.Chart Component Script
[source,javascript]
--
dataset = [["State", "Quantity"], ["NJ", 200], ["NY", 300]]; //<1>
graph = new EGraph();
var elem = new IntervalElement("State", "Quantity");
var titlespec = new TitleSpec(); //<2>
var textspec = new TextSpec(); //<3>
textspec.setColor(java.awt.Color(0xff0000));
titlespec.setLabel("X Title");
titlespec.setTextSpec(textspec); //<4>
graph.setXTitleSpec(titlespec);
graph.addElement(elem);
--
<1> {dataset}
<2> Create the xref:TitleSpec.adoc[] object.
<3> Create the xref:TextSpec.adoc[] object.
<4> Use `setTextSpec()` to assign the `TextSpec` to the title.

[TIP]
=====
To change the property on a Chart that was *previously created* with the Chart Editor, use “getter” methods such as xref:EGraph_getXTitleSpec.adoc[] {circle1} to obtain a handle to the desired xref:TitleSpec.adoc[] object. For example:

[source,javascript]
--
var titlespec = graph.getXTitleSpec(); //<1>
var textspec = new TextSpec();
textspec.setColor(java.awt.Color(0xff0000));
titlespec.setTextSpec(textspec);
--
