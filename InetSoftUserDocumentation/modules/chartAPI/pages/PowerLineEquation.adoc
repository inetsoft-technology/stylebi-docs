= PowerLineEquation
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]



Specifies a power data fit for a Chart.


== Example

.Chart Component Script
[source,javascript]
--
var form = new LineForm(); //<1>
var equation = new PowerLineEquation(); //<2>
form.setLineEquation(equation); //<3>
form.setColor(java.awt.Color(0xff0000));
graph.addForm(form); //<4>
--
<1> Create a  xref:LineForm.adoc[] object.
<2> Create the `PowerLineEquation`.
<3> Use    xref:GraphFormSetLineEquation.adoc[] to apply a the `PowerLineEquation`.
<4> Use xref:EGraphAddForm.adoc[] to add the `LineForm` to the Chart.
