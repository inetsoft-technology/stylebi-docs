= AxisSpec.setLabelOnSecondaryAxis(Boolean)
:navtitle: setLabelOnSecondaryAxis()
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]

Specifies whether axis labels are displayed on the opposite side of the chart. For a Y-axis, labels move from left to right; for an X-axis, labels move from bottom to top. {noGetter}

== Parameter
Boolean:: `true`: display labels on the opposite side +
`false`: display labels on the default side (default)
