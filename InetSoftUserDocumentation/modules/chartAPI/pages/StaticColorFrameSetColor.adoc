[CAUTION,caption=inherits from]
====
[square]
* xref:ColorFrame.adoc[]
====


Specifies the static color to be used for positive field values in element script . The static color can also be passed as an input to the constructor. If the data column specified by the inherited xref:VisualFrameSetField.adoc[] function contains (positive) numbers or colors, these data values are used instead of the `setColor()` value.  You can also set this using the `bindingInfo.colorFrame.color`  property syntax in *onRefresh* script.  The corresponding “getter” function is `getColor()`.



== Paramater (Function Syntax)
value:: a {JavaAwtColor} object

== Example 1


This example illustrates how to use xref:dashboardscript:ChartBindingInfo.adoc[] properties to control the xref:ColorFrame.adoc[].

. Bind a xref:dashboard:BarChart.adoc[]  to the sample ‘All Sales’ Data Worksheet, with ‘Company’ (top 5) on the X-axis, and Sum(Total) on the Y-axis.

[#Example2]
== Example 2

.Chart Component Script
[source,javascript]
----
dataset = [["State", "Quantity"], ["NJ",200], ["NY",300]]; //<1>
graph = new EGraph();
var elem = new IntervalElement("State", "Quantity");
var frame = new StaticColorFrame(); //<2>
frame.setColor(java.awt.Color(0x00ff00)); //<3>
elem.setColorFrame(frame); //<4>
graph.addElement(elem);
----
<1> {dataset}
<2> Create the xref:StaticColorFrame.adoc[] object.
<3> Use `setColor` to specify the  color for the `StaticColorFrame`.
<4> Use xref:GraphElementSetColorFrame.adoc[] to add the `ColorFrame` to the Chart.

[TIP]
=====

To change the property on a Chart that was *previously created* with the Chart Editor, use “getter” methods such as xref:EGraphGetElement.adoc[] {circle1} and xref:GraphElementGetColorFrame.adoc[] {circle2} to obtain a handle to the desired xref:GraphElement.adoc[] and xref:ColorFrame.adoc[]  objects. For example:

[source,javascript]
----
var elem = graph.getElement(0); //<1>
var frame = elem.getColorFrame(); //<2>
// Compact syntax: var frame = graph.getElement(0).getColorFrame();
frame.setColor(java.awt.Color(0xff0000));
----
