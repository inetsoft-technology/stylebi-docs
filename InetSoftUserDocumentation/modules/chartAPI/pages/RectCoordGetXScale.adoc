[square]

====



Retrieves the xref:Scale.adoc[] for the X-axis.   The corresponding “setter” function is xref:RectCoordSetXScale.adoc[].


== Returns
scale::			a xref:Scale.adoc[] object


== Example

.Chart Component Script
[source,javascript]
----
var coord = graph.getCoordinate(); //<1>
var xscale = coord.getXScale(); //<2>
var xspec = xscale.getAxisSpec(); //<3>
// Compact syntax: var xspec = graph.getCoordinate().getXScale().getAxisSpec();
xspec.setLineColor(java.awt.Color(0xff0000)); //<4>


----
<1> Obtain handle to xref:Coordinate.adoc[] object.
<2> Use `getXScale` to obtain a handle to the  xref:Scale.adoc[] object.
<3> Obtain handle to xref:AxisSpec.adoc[] object.
<4> Set desired properties.
