The Upload component’s ‘Properties’ panel provides *General* and *Script* tabs. The next sections discuss the component-specific properties available under these tabs.

== General Tab
In addition to the common properties, the *General* tab in the ‘Upload Properties’ panel provides the following properties.

Submit on Change::	When enabled (default), uploads the selected file immediately. Otherwise, a separate ‘Submit’ button is required.
Label::	Specifies the label to appear on the button.
