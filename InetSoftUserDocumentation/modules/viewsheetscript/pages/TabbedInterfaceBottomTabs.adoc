When `true`, displays the tabs at the bottom of the Tabbed Interface component. When `false` (default), tabs are displayed at the top.

== Type
Boolean:: `true`: tabs appear at the bottom +
`false`: tabs appear at the top (default)


== Example
[source,javascript]
Tab1.bottomTabs = true;




[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
bottomTabs           // unqualified
Tab1.bottomTabs      // qualified

{secondForm}
