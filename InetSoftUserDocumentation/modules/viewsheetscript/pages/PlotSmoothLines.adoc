= smoothLines
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]


For xref:viewsheet:LineChart.adoc[], xref:viewsheet:AreaChart.adoc[], and xref:viewsheet:CircularNetworkChart.adoc[],  set to `true` to render chart lines as smooth curves instead of straight segments.

== Type
Boolean:: `true`: smooth curves (default for  xref:viewsheet:AreaChart.adoc[] and xref:viewsheet:CircularNetworkChart.adoc[]) +
`false`: straight line segments (default for xref:viewsheet:LineChart.adoc[])

== Example
[source,javascript]
Chart1.smoothLines = true;






[TIP]
====
Either syntax below can be used in xref:viewsheetscript:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
smoothLines          // unqualified
Chart1.smoothLines   // qualified

{secondForm}
====
