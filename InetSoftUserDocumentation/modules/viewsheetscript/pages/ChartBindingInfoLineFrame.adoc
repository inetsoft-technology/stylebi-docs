Specifies the aesthetic treatment (line style) for the shape-coded data on a xref:viewsheet:LineChart.adoc[]. A xref:chartAPI:LineFrame.adoc[]  can texture-code chart elements by value, or simply specify a static texture style.




== Type
xref:chartAPI:LineFrame.adoc[]



== Example

[source,javascript]
--
Chart1.bindingInfo.setShapeField("Total",Chart.NUMBER);
Chart1.bindingInfo.lineFrame = new LinearLineFrame;
--
