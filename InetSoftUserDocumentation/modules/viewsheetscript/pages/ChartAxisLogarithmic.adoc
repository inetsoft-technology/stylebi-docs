= axis.logarithmic
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]




Sets logarithmic scaling for a numeric axis.
include::partial$axisShortcut.adoc[]

== Type
Boolean



== Example
[source,javascript]
Chart1.axis['Sum(Product:Total)'].logarithmic = true;

image:commonscript:axisLogarithmic.png[alt="chart with measure axis displayed on logarithmic scale"]



[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
axis['...'].logarithmic             // unqualified
Chart1.axis['...'].logarithmic      // qualified

{secondForm}
====
