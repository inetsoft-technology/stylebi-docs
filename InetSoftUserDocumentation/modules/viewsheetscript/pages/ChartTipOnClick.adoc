= tipOnClick
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]

[WARNING,caption=See Also]
xref:viewsheet:ChartProperties.adoc[] to set this property in the '`Chart Properties`' dialog box.

Specifies, when `true`,  that the data tip view for a Chart is displayed only when the user clicks on the Chart, rather than on hover. If the clicked element has a hyperlink, you must {buttonCtrl}-click to show the data tip.  See xref:viewsheet:AddTipsToChart.adoc[] for more information about data tips.

== Type
Boolean:: `true`: show data tip on mouse click +
`false`: show data tip on mouse hover (default)


== Example
[source,javascript]
Chart1.tipOnClick = true;


[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
tipOnClick         // unqualified
Chart1.tipOnClick  // qualified

{secondForm}
====
