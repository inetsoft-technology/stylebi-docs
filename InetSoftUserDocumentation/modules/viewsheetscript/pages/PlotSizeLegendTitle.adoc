= sizeLegend.title
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]


Sets the legend title for a size-key legend.



== Type
String


== Example

[source,javascript]
Chart1.sizeLegend.title = 'Salesperson';

image:commonscript:sizeLegendTitle.png[alt="chart size legend with custom title text displayed"]

[TIP]
====
Either syntax below can be used in xref:viewsheetscript:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
sizeLegend.title           // unqualified
Chart1.sizeLegend.title    // qualified

{secondForm}
====
