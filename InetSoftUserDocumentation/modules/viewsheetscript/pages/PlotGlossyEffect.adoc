= glossyEffect
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]


Applies a 3D effect to the Chart as a whole. (Not available for all Chart styles.)

== Type
Boolean

== Example
[source,javascript]
Chart1.glossyEffect = true;
