Specifies the text of the label displayed for the Slider component. See xref:viewsheet:SliderSpinnerProperties.adoc[] for information on the '`Label Text`' option.

== Type
String::	Label text


== Example
[source,javascript]
Slider1.labelVisible = true;
Slider1.labelText = 'Select a value';

[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
labelText            // unqualified
Slider1.labelText    // qualified

{secondForm}
