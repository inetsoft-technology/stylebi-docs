Specifies whether a label is displayed for the Slider component. See xref:viewsheet:SliderSpinnerProperties.adoc[] for information on the '`Show Label`' option.

== Type
Boolean:: `true`: label is displayed +
`false`: label is not displayed


== Example
[source,javascript]
Slider1.labelVisible = true;

[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
labelVisible            // unqualified
Slider1.labelVisible    // qualified

{secondForm}
====
