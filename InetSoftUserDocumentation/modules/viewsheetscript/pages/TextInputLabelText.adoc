Specifies the text of the label displayed for the TextInput component. See xref:viewsheet:TextInputProperties.adoc[] for information on the '`Label Text`' option.

== Type
String::	Label text


== Example
[source,javascript]
TextInput1.labelVisible = true;
TextInput1.labelText = 'Enter a value';

[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
labelText               // unqualified
TextInput1.labelText    // qualified
