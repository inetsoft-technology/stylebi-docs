= x2Title.text
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]



Sets the title text for the secondary (top) X-axis.

== Type
string





== Example
[source,javascript]
Chart1.x2Title.text = "New Jersey Employees"




[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
x2Title.text             // unqualified
Chart1.x2Title.text      // qualified

{secondForm}
====
