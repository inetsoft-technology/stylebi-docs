Specifies whether a label is displayed (Combo Box component *only*). See xref:viewsheet:RadioCheckBoxComboProperties.adoc[] for information on the '`Show Label`' option.

== Type
Boolean:: `true`: label is displayed +
`false`: label is not displayed


== Example
[source,javascript]
ComboBox1.labelVisible = true;

[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
labelVisible              // unqualified
ComboBox1.labelVisible    // qualified

{secondForm}
====
