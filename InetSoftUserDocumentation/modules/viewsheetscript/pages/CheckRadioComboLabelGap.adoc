Specifies the gap in pixels between the  component and its label (Combo Box component *only*). See xref:viewsheet:RadioCheckBoxComboProperties.adoc[] for information on the '`Gap (px)`' option.

== Type
Integer::	Gap in pixels between the component and the label (default: 5)


== Example
[source,javascript]
ComboBox1.labelVisible = true;
ComboBox1.labelGap = 10;

[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
labelGap              // unqualified
ComboBox1.labelGap    // qualified

{secondForm}
