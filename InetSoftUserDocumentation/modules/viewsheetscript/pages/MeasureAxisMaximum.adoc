= axis.maximum
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]




Sets the maximum value for a measure axis.
include::partial$axisShortcut.adoc[]

== Type
number






== Example
[source,javascript]
Chart1.axis['Sum(Total)'].maximum = 1500000;

image:commonscript:measureMaximum.png[alt="chart with measure axis maximum value set to 1500000"]



[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
axis['...'].maximum             // unqualified
Chart1.axis['...'].maximum      // qualified

{secondForm}
====
