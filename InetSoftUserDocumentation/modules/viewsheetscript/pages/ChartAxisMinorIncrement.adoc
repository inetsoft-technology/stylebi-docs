= axis.minorIncrement
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]




Sets the numerical increment for numerical  axis tick marks.
include::partial$axisShortcut.adoc[]

== Type
number

== Example
[source,javascript]
Chart1.axis['Sum(Product:Total)'].increment = 300000;
Chart1.axis['Sum(Product:Total)'].minorIncrement = 100000

image:commonscript:measureMinorIncrement.png[alt="chart axis with minor tick marks at 100000 increment intervals"]



[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
axis['...'].minorIncrement             // unqualified
Chart1.axis['...'].minorIncrement      // qualified

{secondForm}
====
