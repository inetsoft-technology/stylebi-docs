= axis.minimum
%experimental%

Sets the minimum value for a numerical  axis.
include::partial$axisShortcut.adoc[]

== Type
number




== Example
[source,javascript]
Chart1.axis['Sum(Product:Total)'].minimum = 50000;

axis['...'].minimum             // unqualified
Chart1.axis['...'].minimum      // qualified
