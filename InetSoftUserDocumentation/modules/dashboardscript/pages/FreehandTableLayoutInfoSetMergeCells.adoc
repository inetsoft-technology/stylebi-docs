Sets the specified cell in a Freehand Table to be merged. See xref:dashboard:AddFreehandTable.adoc[] for more information about ‘Merge Expanded Cells’.




== Parameter
row:: row index
col:: column index
Boolean::			`true` to enable cell expansion +
`false` to disable cell expansion



== Example
[source,javascript]
FreehandTable1.layoutInfo.setMergeCells(1,0,true);

====

FreehandTable1.layoutInfo.setMergeCells  // qualified
