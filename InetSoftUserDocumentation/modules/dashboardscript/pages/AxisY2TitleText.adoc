Summary: The y2Title.text property (type: string) gets or sets the title text for the secondary (right) Y-axis on a Chart component. Used in Dashboard component script, with both qualified (Chart1.y2Title.text) and unqualified (y2Title.text) syntax supported.

Sets the title text for the secondary (right) Y-axis.

== Type
string





== Example
[source,javascript]
Chart1.y2Title.text = "New Jersey Employees"




[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
y2Title.text             // unqualified
Chart1.y2Title.text      // qualified
