This page documents the `doubleCalendar` property of the Calendar component in Dashboard scripts. It is a Boolean property that controls whether the calendar displays in double-panel view (two months or two years) or single-panel view (one month or one year). Set to `true` for double-panel, `false` for single-panel. Can be used as a qualified (`Calendar1.doubleCalendar`) or unqualified (`doubleCalendar`) expression in component script.
endif::ai[]

[WARNING,caption=See Also]
xref:dashboard:CalendarProperties.adoc[] for more information on user selection editing.


Specifies the calendar view type. The single-panel view shows one month or one year (depending on xref:CalendarYearView.adoc[] setting), while the double-panel view shows two months or two years (depending on xref:CalendarYearView.adoc[] setting).

== Type
Boolean::	`true`: double-panel calendar +
`false`: one-panel calendar






== Example
[source,javascript]
Calendar1.doubleCalendar = true;



[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
doubleCalendar               // unqualified
Calendar1.doubleCalendar    // qualified
