Remove the trend line from the measures specified in the array argument. (See xref:dashboard:AddTargetLineTrendLine.adoc[] for information on how to set trend lines.)




== Parameters
array::			An array of measure names for which to hide the trend line




== Examples

[source,javascript]
Chart1.setTrendLineExcludedMeasures(['Sum(Product:Total)','Count(Order:Num)'])
