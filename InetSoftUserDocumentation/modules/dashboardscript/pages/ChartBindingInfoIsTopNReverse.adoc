Returns the state of the Top-N/Bottom-N filter.







== Parameter
col::			Name of the filtered field



== Return
Boolean:: `true`: filter is bottom N +
`false`: filter is top N









== Example

[source,javascript]
--
var rev = Chart1.bindingInfo.isTopNReverse("Company");
