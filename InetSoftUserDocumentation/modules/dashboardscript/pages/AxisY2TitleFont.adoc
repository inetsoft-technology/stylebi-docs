Summary: Sets the title font for the secondary (right) Y-axis title on a Chart component in Dashboard script. Accepts a java.awt.Font object or a font description string (e.g., 'Lucida Sans Unicode-BOLD-14'). Accessed as y2Title.font (unqualified in component script) or Chart1.y2Title.font (qualified).
endif::ai[]


Sets the title font for the secondary (right) Y-axis.


== Type
{javaawtfont} or font description

== Example
[source,javascript]
Chart1.y2Title.font = new java.awt.Font('Serif', java.awt.Font.BOLD, 12);
Chart1.y2Title.font = 'Lucida Sans Unicode-BOLD-14';


[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
y2Title.font            // unqualified
Chart1.y2Title.font     // qualified

{secondForm}
====
