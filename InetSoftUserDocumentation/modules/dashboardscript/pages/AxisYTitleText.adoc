= yTitle.text

// metadata for AI
ifdef::ai[]
Summary: The `yTitle.text` property (string) gets or sets the Y-axis title text on a Chart component in Dashboard script. It can be used in component script as either `yTitle.text` (unqualified) or `Chart1.yTitle.text` (qualified).
endif::ai[]



Sets the Y-axis title text.

== Type
string





== Example
[source,javascript]
Chart1.yTitle.text = "New Jersey Employees"




[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
yTitle.text             // unqualified
Chart1.yTitle.text      // qualified

{secondForm}
====
