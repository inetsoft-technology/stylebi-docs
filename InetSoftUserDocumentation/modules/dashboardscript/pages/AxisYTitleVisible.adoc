Summary: Documents the `yTitle.visible` Boolean property for Chart components in Dashboard script. Gets or sets the visibility of the Y-axis title.

Sets the visibility of the Y-axis title.

== Type
Boolean






== Example
[source,javascript]
Chart1.yTitle.visible = false;




[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
yTitle.visible             // unqualified
Chart1.yTitle.visible      // qualified

{secondForm}
