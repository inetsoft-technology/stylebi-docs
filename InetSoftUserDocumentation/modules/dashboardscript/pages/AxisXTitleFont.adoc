// metadata for AI
ifdef::ai[]
Summary: The xTitle.font property sets the X-axis title font for a Chart component in Dashboard script. Accepts a java.awt.Font object or a font description string (e.g., 'FontName-STYLE-SIZE'). It is a writable property accessed as Chart1.xTitle.font (qualified) or xTitle.font (unqualified in component script).
endif::ai[]


Sets the X-axis title font.




== Type
{javaawtfont} or font description

== Example
[source,javascript]
Chart1.xTitle.font = new java.awt.Font('Serif', java.awt.Font.BOLD, 12);
Chart1.xTitle.font = 'Lucida Sans Unicode-BOLD-14';


[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
xTitle.font            // unqualified
Chart1.xTitle.font     // qualified

{secondForm}
====
