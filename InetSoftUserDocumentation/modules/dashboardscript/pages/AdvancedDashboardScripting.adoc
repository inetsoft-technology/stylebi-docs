= Advanced Dashboard Scripting
:page-aliases: viewsheetscript:AdvancedDashboardScripting.adoc

Certain Dashboard applications require a degree of flexibility that can only be achieved with programmatic logic. For example, you may need to hide or show certain Dashboard components under particular conditions, such as when a user’s selections return no data, or you might wish to present an alert message if a certain key indicator does not meet its target.  The Dashboard scripting environment allows you to easily implement such event-driven behavior.

TIP: The scripting language is based on the popular JavaScript language  (https://www.ecma-international.org/publications/standards/Ecma-262.htm[ECMA-262] standard).


The following sections provide a short introduction to JavaScript, concentrating on the Dashboard environment and examples of common Dashboard scripting. These sections are intended for advanced Dashboard designers who are familiar with programming concepts and languages.

[square]
** xref:dashboardscript:IntroToProgramming.adoc[]: Basic JavaScript programming syntax and programming constructs such as loops, variables, and functions.
** xref:dashboardscript:AddScriptToDashboard.adoc[]: Describes the different places that you can add script to a Dashboard, at the  Dashboard-level, component-level, or  property-level.
** xref:dashboardscript:CreateScriptFunction.adoc[]: Create a reusable script function that can be accessed in any Dashboard.
** xref:dashboardscript:AccessComponentData.adoc[]: Access component values, such as Filter values, Table cell values, or KPI values, within a scripted expression.  This allows you to adapt  Dashboard properties or appearance in response to displayed  data.
** xref:dashboardscript:AccessDataSourceData.adoc[]: Access and aggregate data from the Dashboard data source using scripted expressions.
** xref:dashboardscript:RunQueryFromScript.adoc[]:  Retrieve data from an arbitrary Data Worksheet using a scripted expression.
** xref:dashboardscript:ReferenceQueryData.adoc[]: Access the data retrieved by the `runQuery` function in script.   These techniques are mainly useful in Freehand Tables (see xref:dashboard:AddFreehandTable.adoc[]).
** xref:dashboardscript:SpecialFreehandFunctions.adoc[]: Special functions that are mainly useful when scripting a Freehand Table.
** xref:dashboardscript:ChangeImageWithScript.adoc[]:  Use script to dynamically change the image in an Image component .
** xref:dashboardscript:DashboardObjectReference.adoc[]: Reference descriptions of all Dashboard properties that can be modified in script.
