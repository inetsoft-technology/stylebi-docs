Summary: The `fields` property of a Calendar component specifies the data block field to be filtered by the Calendar. It is an array type and is equivalent to setting the field under the Data tab of the component Properties dialog. Used in Dashboard component script (qualified as `Calendar1.fields` or unqualified as `fields`).
endif::ai[]

[WARNING,caption=See Also]
xref:dashboardscript:CalendarQuery.adoc[] to specify the data block from which to draw the fields.



Specifies the field of the data block to be filtered by the Calendar component. This is equivalent to selecting the field under the *Data* tab of the component ‘Properties’ dialog box.

image::dashboard:calendarFields.png[alt="calendar component Data tab showing fields property binding"]

== Type
array



== Example
[source,javascript]
Calendar1.query = 'Sales';
Calendar1.fields = ['Date'];




[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
fields               // unqualified
Calendar1.fields    // qualified

{secondForm}
====
