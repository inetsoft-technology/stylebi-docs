Summary: The `x2Title.visible` property on a Chart component sets or gets the visibility of the secondary (top) X-axis title in Dashboard component script. It is a Boolean. The default is `true` (visible) if a value has been provided for x2Title.text, otherwise `false`.
endif::ai[]



Sets the visibility of the secondary (top) X-axis. The default is `true` (visible) if a value has been provided for xref:AxisX2TitleText.adoc[], otherwise `false`.

== Type
Boolean



== Example
[source,javascript]
Chart1.x2Title.visible = false;




[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
x2Title.visible             // unqualified
Chart1.x2Title.visible      // qualified

{secondForm}
====
