= pieExploded
:page-aliases: viewsheetscript:PlotPieExploded.adoc
%experimental%
include::ROOT:partial$buttonSnippets.adoc[]


Separates the slices of a xref:dashboard:PieChart.adoc[] by a small gap.



== Type
Boolean:: `true`: explode pie +
`false`: do not explode pie

== Example
[source,javascript]
Chart1.pieExploded = true


image:commonscript:pieExploded.png[alt="pie chart with one or more slices exploded outward"]



[TIP]
====
Either syntax below can be used in xref:dashboardscript:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
pieExploded          // unqualified
Chart1.pieExploded   // qualified

{secondForm}
====
