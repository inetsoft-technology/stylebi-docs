// metadata for AI
ifdef::ai[]
Summary: Sets or gets the font of the Y-axis title on a Chart component. Accepts a java.awt.Font object or a font description string (e.g., 'Lucida Sans Unicode-BOLD-14'). Available in Dashboard component script as yTitle.font (unqualified) or Chart1.yTitle.font (qualified).
endif::ai[]


Sets the Y-axis title font.



== Type
{javaawtfont} or font description

== Example
[source,javascript]
Chart1.yTitle.font = new java.awt.Font('Serif', java.awt.Font.BOLD, 12);
Chart1.yTitle.font = 'Lucida Sans Unicode-BOLD-14';


[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
yTitle.font            // unqualified
Chart1.yTitle.font     // qualified

{secondForm}
====
