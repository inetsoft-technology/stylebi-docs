Summary: For Date/Time and Categorical shape legends, set to `true` to suppress the display of `null` dimension values in the legend. When `false` (default), `null` values are represented without a label in the legend.

For Date/Time and Categorical shape legends, set to `true` to suppress the display of `null` dimension values in the legend. When `false` (default), `null` values are represented without a label in the legend.



== Type
Boolean


== Examples

[source,javascript]
Chart1.shapeLegend.noNull = true;



[TIP]
====
Either syntax below can be used in xref:dashboardscript:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
shapeLegend.noNull          // unqualified
Chart1.shapeLegend.noNull   // qualified
