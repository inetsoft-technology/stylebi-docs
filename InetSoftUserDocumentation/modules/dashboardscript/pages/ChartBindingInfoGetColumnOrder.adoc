Returns the sorting order for a specified dimension.



== Parameter
col::			Name of field for which to obtain sorting

== Return
Number:: 1=ascending +
2=descending +
0=none


== Example
[source,javascript]
--
var ord = Chart1.bindingInfo.getColumnOrder("Company")
