Sets the corner radius for nodes on a xref:dashboard:TreeChart.adoc[]. A value of `0` produces square corners; `0.5` produces the maximum rounding. Default is `0.3`.

== Type
Number:: A value between `0` (square) and `0.5` (maximum rounding)

== Example
[source,javascript]
Chart1.nodeCornerRadius = 0.5;
