Summary: Documents the xTitle.visible property for Dashboard chart component scripts. This Boolean property gets or sets the visibility of the X-axis title on a Chart component.
endif::ai[]



Sets the visibility of the X-axis title.

== Type
Boolean






== Example
[source,javascript]
Chart1.xTitle.visible = false;




[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
xTitle.visible             // unqualified
Chart1.xTitle.visible      // qualified

{secondForm}
====
