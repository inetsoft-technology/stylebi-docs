Specifies an image file either by an uploaded image name (e.g., `'image1.gif'`), a resource path on the server (e.g., `'/library/image1.gif'`), a URL (e.g., `“https://www.google.com/images/logos/ps_logo2.png”`), or a BLOB stored in a data block. See xref:dashboard:ImageProperties.adoc[] for information on uploading images.




== Type
String:: name of image (e.g., `'/myImage.jpg'`)





== Example
[source,javascript]
Image1.image = '/library/image1.gif';

[TIP]
====
Either syntax below can be used in  xref:AddScriptToDashboard.adoc#AddComponentScript[component script]:
[source,javascript]
image           // unqualified
Image1.image    // qualified

{secondForm}
====
