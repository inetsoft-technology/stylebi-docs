Specifies the aesthetic treatment for shape-coded data on a xref:dashboard:PointChart.adoc[]. A xref:chartAPI:ShapeFrame.adoc[] can shape-code chart elements by value, or simply specify a static shape style.






== Type
xref:chartAPI:ShapeFrame.adoc[]



== Example

[source,javascript]
----
Chart1.bindingInfo.setShapeField("Total",Chart.NUMBER);
Chart1.bindingInfo.shapeFrame = new TriangleShapeFrame;
----
