InetSoft StyleBI  allows you to explore, analyze, monitor, report, and collaborate on business and operational data, all in real time.  StyleBI is engineered to scale with your enterprise needs, and to seamlessly integrate with a wide variety of data sources and delivery platforms.

== About This Documentation

This user documentation explains how to use the User Portal to create data sources, Data Worksheets, and Dashboards, as well as how to manage the server with Enterprise Manager.  It contains the following main modules:

[square]
* xref:user:ConnectToData.adoc[]: Add a  *datasource* to provide a persistent connection to your data, so that changes that you make to your data by updating, modifying, or deleting records are automatically reflected in all Dashboards that use the data source.

* xref:dataworksheet:PrepareYourData.adoc[]: Create  a *Data Worksheet*  using *Visual Composer* to manipulate your dataset (e.g.,  filter, group, sort, transform) or  mashup data from multiple data sources.

* xref:dashboard:VisualizeYourData.adoc[]: Create a *Dashboard* using  *Visual Composer*  to dynamically filter a data set and visualize the output on tables, charts, and gauges.

* xref:user:ConfigureSelfService.adoc[]: Configure self-service features to enable users to most effectively create their own  dashboards.

* xref:dashboard:PublishDashboard.adoc[]: Arrange components on a Dashboard for publication across a variety of devices and languages.

* xref:dashboard:DashboardReference.adoc[]: Descriptions of all properties available in Dashboard component dialog boxes.

* xref:dashboardscript:AdvancedDashboardScripting.adoc[]: Add    flexibility to Dashboard applications with programmatic logic.

* xref:administration:ManageServer.adoc[]: Use Enterprise Manager  to manage the InetSoft server with features such as monitoring, logging, configuration, file management, and security.

* xref:commonscript:UserFunctions.adoc[]: Brief introduction to common JavaScript language constructs and typical usage.

* xref:chartAPI:ChartScriptTutorial.adoc[] and xref:chartAPI:ChartScriptReference.adoc[]:  Examples and reference pages for the Chart API, which allows you to access deep Chart functions to create custom Charts using scripted logic.

* xref:user:UsePortal.adoc[]: Navigate the User Portal as an end-user, and perform common Dashboard functions such as exporting, bookmarking, annotating, and sharing.
