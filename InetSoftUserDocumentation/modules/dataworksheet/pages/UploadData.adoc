In most cases, data sources will already be configured for you by an administrator. However, you can also import a plain text file, CSV file, or Excel file into a Data Worksheet. Follow the steps below to import a file into a Data Worksheet.



include::ROOT:partial$createVisualizationStep1Upload.adoc[]

. Create a new Data Worksheet and select the {buttonUploadFile} option. (See xref:CreateNewWorksheet.adoc[].) This opens the ‚ÄòImport Data File‚Äô dialog box.
+
image:dataworksheet:embeddedTable4.png[alt="Import Data File dialog with encoding delimiter and header options"]
+
If a Data Worksheet is already open in Visual Composer, press the {buttonCreateObject} on the toolbar, and select {buttonUploadFile} to open the ‚ÄòImport Data File‚Äô dialog box.
+
image:textSource5_1.png[alt="press Create Object button and select Upload File to open Import Data File dialog"]

. Press the btn:[Choose File] or btn:[Browse] button to open the ‚ÄòOpen File‚Äô dialog box.
In this example, we will import an Excel file: Select an Excel file type (`.xls`, `*.xlsx`) from the menu next to the ‚ÄòFile name‚Äô field.
+
image:uploadFile2.png[alt="Open File dialog with Excel file type filter selected"]

. Select the desired Excel file and press btn:[Open]. (Here we use the sample xref:ROOT:attachment$OrderDataText.xls[OrderDataText] Excel file.)
+
image:uploadFile1.png[alt="select the OrderDataText Excel file and press Open"]

. Press btn:[OK] to import the data. (For the sample _OrderDataText.xls_ Excel file, you do not need to change the other settings.) This creates a new data block  named ‚ÄòOrderDataText‚Äô with data from the file.
+
image:embeddedTable5.png[alt="new OrderDataText data block created with imported file data"]

. Right-click on the data block, and select ‚ÄòProperties‚Äô from the context menu.  {rightClickOptions} This opens the ‚ÄòTable Properties‚Äô panel.

. In the ‚ÄòName‚Äô field, enter  a name for the data block, and press btn:[OK].
+
image:embeddedTable7.png[alt="Table Properties panel with Name field for renaming the data block"]
+
include::ROOT:partial$duplicateNames.adoc[]


. If you want to manually modify the data in the data block, convert the data block into an Embedded Table. See xref:EnterData.adoc[] for more information.

. Press the {buttonSave} in the toolbar to save the Data Worksheet.
