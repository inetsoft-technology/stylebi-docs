= Edit a Data Worksheet
include::ROOT:partial$buttonSnippets.adoc[]
include::ROOT:partial$rightClickOptions.adoc[]

To edit a Data Worksheet in the User Portal, follow the steps below.

include::ROOT:partial$createVisualizationClose.adoc[]

. Press the top {buttonDataTab} in the Portal.
+
image:user:uploadDriver1.png[alt="press the Data tab at the top of the Portal"]
. Click to expand the folder containing the Data Worksheet that you want to edit.
+
image:user:PortalEditWorksheet1.png[alt="expand folder in Portal to find the Data Worksheet to edit"]
. Click the name of the Data Worksheet that you want to edit.
+
image:user:PortalEditWorksheet2.png[alt="click the name of the Data Worksheet to open it for editing"]
+
This opens the Data Worksheet for editing in Visual Composer. You can now do data pipeline operations on this Data Worksheet, as described in the following sections (xref:dataworksheet:FilterData.adoc[], xref:dataworksheet:JoinData.adoc[], etc.).


If Visual Composer is already open, you can open a Data Worksheet for editing by locating the Data Worksheet in the left panel. Double-click the Data Worksheet, or right-click and select ‘Open Sheet’ from the menu. Alternatively, you can drag the Data Worksheet into the canvas on the right side.

image:dataworksheet:editWorksheetPortal.png[alt="double-click or right-click Data Worksheet in left panel to open it"]
