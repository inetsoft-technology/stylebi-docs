To access Data Worksheet properties, press the {buttonOptions} in the Visual Composer toolbar.

image::dataWorksheetOptions.png[alt="press Options button in the Visual Composer toolbar"]

Enter a desired  '`Alias`' and '`Description`' for the Data Worksheet.  When  specified, the alias will be displayed in the data source listing instead of the name.

image::dataWorksheetOptions2.png[alt="Data Worksheet options panel showing alias and description fields"]

[TIP]
====
To *change the name* of a Data Worksheet, press the {buttonActions} next to the desired Data Worksheet
====
