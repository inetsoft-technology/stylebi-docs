To access Data Worksheet properties, press the {buttonOptions} in the Visual Composer toolbar.

image:dataWorksheetOptions.png[]

Enter a desired  '`Alias`' and '`Description`' for the Data Worksheet.  When  specified, the alias will be displayed in the data source listing instead of the name.
