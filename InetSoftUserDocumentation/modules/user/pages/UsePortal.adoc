The User Portal provides features for accessing and managing existing assets, such as Data Worksheets and Dashboards, and allows you to create new assets as well. The User Portal is a web-based interface, so you do not need to install any special software on your computer. Your system administrator will tell you the correct address to use in your web browser.


The following sections explain how to perform common tasks in the User Portal:

[square]
** xref:user:OpenUserPortal.adoc[]: Access the User Portal in your browser.
** xref:user:ConfigurePortal.adoc[]: Change your password and email.
** xref:user:ManageRepository.adoc[]: Create folders and organize your Dashboards.
** xref:user:UseDashboard.adoc[]: Understand Dashboard toolbar features.
** xref:viewsheet:UseFilterComponents.adoc[]: Use Filter components to explore your data and generate actionable data  insights.
** xref:viewsheet:UseCharts.adoc[]: Use Chart features such as zoom, show details, brushing, and ad hoc editing.
** xref:user:BookmarkDashboard.adoc[]: Save Dashboard filters, annotations, and other settings in a bookmark that can be reloaded at a later time.
** xref:viewsheet:AddAnnotation.adoc[]: Add notes to Dashboard components or data points.
** xref:user:ProfileDashboard.adoc[]: Investigate the performance of a Dashboard to determine the source of Dashboard loading delays.
** xref:user:ScheduleDashboard.adoc[]: Configure a Dashboard for automatic generation, delivery to users, and alerts.
** xref:user:DeployDashboard.adoc[]: Pin a Dashboard to the Portal for quick and frequent access.
** xref:viewsheet:ShareDashboard.adoc[]: Share a Dashboard using social networks or  email.
