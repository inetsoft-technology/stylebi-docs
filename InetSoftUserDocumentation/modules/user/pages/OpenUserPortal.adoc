How to open the User Portal from the main landing page, including logging in with username, password, and optional Locale. Describes the main navigation options in the Portal: Home (return to landing page), Dashboard (access dashboards), Repository (view all available dashboards), Create (create dashboards, data worksheets, or open Visual Composer), Schedule (schedule dashboard generation and monitor tasks), Data (create and manage data sources), Help (open documentation), and Account (access Preferences or Logout). Available options depend on permissions granted by the administrator.
endif::ai[]



To open the User Portal, follow the steps below:


. On the main landing page, press the  '`Dashboard Portal`' link.

+
image::StyleBILandingPage.png[alt="press Dashboard Portal link"]

. Login to the server.  If security is enabled on the server, you must log in before you can use the Portal. Enter your username and password in the provided ‘User’ and ‘Password’ fields. (Ask your system administrator if you don’t know your username and password.) If you are prompted to enter a ‘Locale’, select the correct option for your location.
+
image::user:DefaultUserLogin.png[alt="enter username and password to log in"]
+
This opens the User Portal.
+
image::InetSoft Portal Tabs.png[alt="User Portal opens"]
+
The specific options available in the User Portal will depend on what permissions you have been granted by the administrator.
+
include::ROOT:partial$createVisualizationClose.adoc[]


. Select the desired option in the User Portal.
+
****


{buttonHome}:: Return to the InetSoft landing page.
{buttonDashboard}:: Access your Pinboards — Dashboards you've added to this tab for quick, repeated access (see xref:ROOT:Glossary.adoc[Glossary]). See xref:DeployDashboard.adoc[] for information on configuring Pinboards.
+
[TIP]
====
If you are using the Portal for the first time, you can also press the Pinboards link to deploy your first Dashboard.

image::UserPortalFirstTimePortalDashboards.png[alt="press Pinboards link"]
====
{buttonRepository}:: View all available Dashboards in the repository.  See xref:ManageRepository.adoc[] for information on how to organize the Repository.
{buttonCreate}:: Create data sources, Data Worksheets, and Dashboards, or open  Visual Composer. The following options are available for  users who have appropriate permissions:
+
--
[square]
* {buttonCreateDashboard}: Create a new Dashboard. See xref:dashboard:CreateNewDashboard.adoc[] for more information.
* {buttonWorksheet}: Create a new Data Worksheet. See xref:dataworksheet:CreateNewWorksheet.adoc[] for more information.
* {buttonLaunchComposer}: Open  Visual Composer in a new browser tab.
--
{buttonSchedule}:: Schedule Dashboards to generate automatically, and monitor queued and completed scheduled tasks. See xref:user:ScheduleDashboard.adoc[]  for more details.
{buttonData}:: Create and manage data sources. See xref:ConnectToData.adoc[] for more information.
{buttonHelp}:: Open this documentation.
{buttonAccount}:: Provides access to {buttonPreferences}, where you can change personal settings and configure private Dashboards, and provides the option to {buttonLogout} of the Portal (when security is enabled). See xref:ConfigurePortal.adoc[] for more information.
****
