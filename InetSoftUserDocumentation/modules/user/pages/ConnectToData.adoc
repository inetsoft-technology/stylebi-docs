= Connect to Your Data

A *datasource* provides a persistent connection to your data, so changes that you make to your data by updating, modifying, or deleting records are automatically reflected in all Dashboards that use the data source.

== Overview
This module contains the following key sections, and is relevant to administrators, designers, and end-users:

[square]

* xref:ConnectToDataSource.adoc[]: Find your database in the listing provided in this section, and follow the instructions to create a *new datasource*.  If you are using a relational database, also see xref:NeedDataModel.adoc[]

* xref:dataworksheet:UploadData.adoc[]: Import data from a text file, CSV file, or Excel file directly into a Data Worksheet. This is  useful if you do not wish to create a persistent datasource.

* xref:EditConnection.adoc[]: Edit an existing datasource connection.

* xref:AddDataSecurity.adoc[]: Add data-level security that restricts available data based on user, group, role, or any other parameters and conditions.
