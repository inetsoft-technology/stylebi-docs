include::partial$dataSourceUserIntro.adoc[]

. Select the ‘Firebase’ data source, and press btn:[Create]. This opens the data source definition screen.
+
image::dataSourceNewFirebase2.png[alt="Firebase data source definition screen"]

. In the ‘Name’ field, enter a label for this data source connection.


. Enter the ‘Project ID’  and press btn:[Authorize]. This will populate the token fields.
+
. *_Optional:_* To refresh meta-data for the data source (which may be necessary if the data source's fields or structure have changed), press the btn:[Refresh Metadata] button.

. *_Optional:_* {multiTenantUser}

. Press btn:[OK].

+
include::ROOT:partial$createVisualizationStep2.adoc[]

. Verify that the data source you added is shown with a {buttonCheckIcon}, indicating that the connection was successful.
+
image::dataSourceFirebase2.png[alt="connection successful icon"]
+
If the database is shown with an {buttonError}, this means that the connection was not successful.
+
image::dataSourceFirebase3.png[alt="connection error icon"]
+
In this case, check the settings to make sure they are correct for your database. See xref:EditConnection.adoc[] for information on how to change connection settings.

After you have created a connection to the database, you can proceed to define a data set. See xref:dataworksheet:QueryFirebase.adoc[].
