How to deploy, edit, delete, and reposition Pinboards in the User Portal, including selecting which Dashboard to display under each tab, adjusting the tab order, and hiding Dashboard tabs using the Arrange Dashboards dialog.

You can arrange the Pinboards displayed by the {buttonDashboard} of the User Portal, and add your own Pinboards under this button.

image::user:Opening Sample Viewsheet2.png[alt="Pinboards in User Portal"]

== Deploy a Pinboard

To deploy a Dashboard, follow the steps below:

. Press {buttonDashboard} the  in the User Portal.

. Press the {buttonDashboardConfiguration}, and select the  {buttonAdd} option.
+
image::user:Dashboard Button.png[alt="press Dashboard Configuration and select Add"]
+
This opens the ‘New Pin’ dialog box.
+
image::user:newDashboard.png[alt="New Pin dialog box"]

. In the ‘Dashboard Name’ field, enter a name for the new Dashboard. This name will be displayed on the Dashboard’s tab in the Portal.

. *_Optional:_*  Enter a description for the Dashboard in the ‘Description’ field. This description is only visible when the Dashboard is being edited.

. Choose the desired Dashboard from the ‘Select Dashboard’ list.
+
image::Edit Dashboard User VS.png[alt="choose Dashboard from Select Dashboard list"]

. Press btn:[OK]. This deploys the Dashboard as a Pinboard.
+
image::user:My Viewsheet Dashboard.png[alt="press OK to deploy Pinboard"]




== Edit a Pinboard
To edit an existing Pinboard, follow the steps below:

. Press  the {buttonDashboard} in the User Portal.

. Open the desired Pinboard by selecting the corresponding tab.

. Press the {buttonDashboardConfiguration}, and select the {buttonEditAction} option.
+
image::user:Edit Dashboard User.png[alt="press Dashboard Configuration and select Edit"]
+
This opens the ‘Edit Dashboard’ dialog box, which is the same as the ‘New Dashboard’ dialog box described above. Follow the instructions above to make the desired edits to the Dashboard name and description, or to select a new Dashboard to display.

== Delete a Pinboard
To delete an existing Pinboard, follow the steps below:

. Press the {buttonDashboard} in the User Portal.

. Open the desired Pinboard by selecting the corresponding tab.

. Press the {buttonDashboardConfiguration} , and select the {buttonDeleteAction} option.
+
image::user:deleteDashboard.png[alt="press Dashboard Configuration and select Delete"]

== Reposition a Pinboard
To reposition an existing Pinboard, follow the steps below:

. Press the {buttonDashboard} in the User Portal.

. Open the desired Pinboard by selecting the corresponding tab.

. Press the {buttonDashboardConfiguration}, and select the {buttonArrangeAction}  option. This opens the ‘Arrange Dashboards’ dialog box.
+
image::user:arrangeDashboard.png[alt="press Dashboard Configuration and select Arrange"]

. Press the ‘up’ or ‘down’ arrow in the ‘Arrange Order’ column to alter the Dashboard’s position in the sequence of Portal tabs. Uncheck the ‘Enable’ option to hide the Dashboard’s tab.
+
image::user:arrangeDashboard2.png[alt="press up or down arrow to reposition Dashboard"]

. Press btn:[OK] to close the dialog box.
