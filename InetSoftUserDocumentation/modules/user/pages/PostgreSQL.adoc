. Select the ‘PostgreSQL’ data source, and press btn:[Create]. This opens the data source definition screen.
+
image:dataSourceNewPostgreSQL2.png[]

. In the ‘Name’ field, enter a label for this database connection.

. From the ‘Database Type’ menu, select ‘PostgreSQL’ (if not already selected).
+
NOTE:  The PostgreSQL database driver is packaged with InetSoft. However, if you see a warning that the driver is not available, contact your administrator.


. If your administrator has provided a connection URL, enable ‘Custom URL’, and enter the  URL in the '`URL`' field. It should have the following form:
+
 jdbc:postgresql://{host}:{port}/{databaseName}
+
*If you do not have a URL*, disable ‘Custom URL’, and enter the additional required connection information:  ‘Host Name’, ‘Port’, ‘Database Name’, and login credentials.
