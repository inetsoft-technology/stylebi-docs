. Select the ‘Data.Gov JSON’ data source, and press btn:[Create]. This opens the data source definition screen.
+
image::dataSourceDataGov2.png[alt="Data.Gov JSON data source definition screen"]

. In the ‘Name’ field, enter a label for this data source connection.


. Enter the  ‘URL’ for the web service and any required credentials.
+
. *_Optional:_* {multiTenantUser}

. Press btn:[OK].

. Verify that the data source you added is shown with a {buttonCheckIcon}, indicating that the connection was successful.
+
image::dataSourceDataGov3.png[alt="connection successful icon"]
+
If the database is shown with an {buttonError}, this means that the connection was not successful.
+
image::dataSourceDataGov4.png[alt="connection error icon"]

In this case, check the settings to make sure they are correct for your database. See xref:EditConnection.adoc[] for information on how to change connection settings.

After you have created a connection to the database, you can proceed to define a data set. See xref:dataworksheet:QueryDataGovJSON.adoc[].
