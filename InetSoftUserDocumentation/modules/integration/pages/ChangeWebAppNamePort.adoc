How to change the StyleBI web application port by modifying the ports setting in docker-compose.yaml. The default port is 8080. Important: the server must be stopped before making changes, and the container must be recreated for the port change to take effect.
endif::ai[]

IMPORTANT: Stop your server before making these changes. The container needs to be recreated for the port change to take effect.

The default URL for the repository is http://hostname:8080/app/portal/tab/report, where  `8080` is the default port number. To change the webapp port, modify the `ports` setting in the _docker-compose.yaml_ file, as shown below.



.Example (YAML snippet): Assign port 80
[source,yaml]
server:
    image: "${STYLEBI_DOCKER_IMAGE:-ghcr.io/inetsoft-technology/stylebi-community:latest}"
    restart: unless-stopped
    ports:
      - "80:8080"
